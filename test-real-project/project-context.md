# Proyecto de Prueba - Landing Page Simple

## Descripción
Crear una landing page HTML simple con CSS básico.

## Stack
- HTML5
- CSS3 (vanilla)
- Sin frameworks

## Estructura Esperada
```
project/
├── index.html
├── styles.css
└── README.md
```

## Requisitos
- Header con navegación
- Hero section con título y subtítulo
- Footer
- Responsive (mobile-first)

## Convenciones
- Usar nombres de clases BEM
- CSS en archivo separado
- HTML semántico
