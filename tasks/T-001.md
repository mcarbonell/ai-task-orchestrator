## Descripción
Necesitamos crear una página web básica llamada `landing.html` en el directorio raíz. 
La página debe contener:
1. Un `<h1>` con el texto exacto: "Bienvenido al Orquestador V2"
2. Un `<button>` llamativo con el texto exacto: "Iniciar Sistema"
3. Estilos CSS modernos (fondo oscuro `#1e1e1e`, texto claro, y el botón con algo de color y estilo redondeado) aplicados en la cabecera del archivo.

La idea es probar que el agente es capaz de leer esta instrucción, crear el HTML, pasar los tests de la terminal y luego el CDP Wrapper navegará al archivo para sacar la captura de pantalla que será validada por la propia IA.

## Criterios de Aceptación
- [ ] Se crea el archivo `landing.html`
- [ ] El título contiene "Bienvenido al Orquestador V2"
- [ ] Hay un botón "Iniciar Sistema"
- [ ] Contiene estilos básicos.
